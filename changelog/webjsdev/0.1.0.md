## Features

- **initial release** The webjs CLI, published unscoped on npm so `npx webjsdev <cmd>` works without typing the `@webjsdev/` scope. Installs three commands from one package: the canonical `webjs`, the short `wjs`, and the package-name `webjsdev` itself, all pointing at the same one-line ESM re-export of `@webjsdev/cli/bin/webjs.js`. No argv manipulation, no spawn overhead, behaviour matches `@webjsdev/cli` exactly.
  The original name choice was `wjs`, but npm blocked that for typosquatting similarity with `w-js` and `w.js`. `webjsdev` is the same shape with the org name as the package name, which both reads as the canonical brand and clears the similarity check.
