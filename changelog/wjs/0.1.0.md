## Features

- **initial release** Short alias for the `@webjsdev/cli` `webjs` binary, so `npx wjs <cmd>` works the same as `npx @webjsdev/cli <cmd>` or a global `webjs <cmd>`. The package is a tiny ESM re-export of `@webjsdev/cli/bin/webjs.js`, with no argv manipulation or spawn overhead, so behaviour stays identical between `webjs <cmd>` and `wjs <cmd>`.
  The package declares both `wjs` and `webjs` in its `bin` map, so `npm i -g wjs` installs **two** global symlinks: the short alias plus the canonical `webjs` name that the framework's docs and scaffold templates use. Users who want only the short name can ignore the `webjs` symlink; users who want their daily command to match the documentation can use `webjs <cmd>`.
