## Fixes

- **scaffold's next-steps banner now references the scoped CLI for `webjs ui`** Previous output suggested `npx webjs ui add <name>`, but the bare `webjs` npm name is owned by an unrelated package. Outside a scaffolded project with the local node_modules in PATH, `npx webjs <cmd>` would fetch THAT package rather than ours. Switched the two banner lines to the explicit scoped form `npx @webjsdev/cli ui ...`, which is unambiguous everywhere.
