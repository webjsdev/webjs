## Fixes

- **detect module-scope client work by an allowlist of safe top-level forms**
  Elision's "does this module do client work at load" check is an allowlist
  (top level may only declare and register; any other top-level call, `new`,
  dynamic `import()`, or `await` ships) instead of a denylist of browser
  global names. This closes false-elision holes (a module-scope `fetch`,
  `new WebSocket`, or dynamic `import()` was wrongly elided) and does not rot
  as browsers add APIs.
